## [download](#)

A action download permite realizar o download de qualquer tipo de arquivo a partir de uma url

### [import](#)

```javascript
import download from '@kuba/download'
```

### [uso](#)

Passe a url do arquivo que deseja baixar pela action

```javascript
download(url)
```
