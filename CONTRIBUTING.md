## Como Contribuir?
  <img src="https://github.com/oneOffJS/rex.community/raw/master/rex.png" alt="Rex Logotipo" width="300" height="300" />

  - Fork o projeto
  - Crie seu branch de recursos: git checkout -b my-new-feature
  - Faça commit de suas alterações: git commit -m 'Adicionar algum recurso'
  - Envie para o branch: git push origin my-new-feature
  - Lembre-se que temos um pré-push hook com etapas que analisa e evita erros.
  - Depois que sua solicitação pull for mesclada, você pode excluir seu branch com segurança.
